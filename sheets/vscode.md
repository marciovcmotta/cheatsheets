| Comment/uncomment line | Ctrl + / | Cmd + / |
| Indent line | Tab | Tab |
| Outdent line | Shift + Tab | Shift + Tab |

| Go to file | Ctrl + P | Cmd + P |
| Go to symbol | Ctrl + Shift + O | Cmd + Shift + O |

| Go to definition | F12 | F12 |

| Go forward | Alt + Right Arrow | Ctrl + Shift + - |
